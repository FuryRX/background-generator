## HTML ##
You first want to have a rough idea of your website.
Split it up into
* Header
* Navigation 
* Main Content
* Sidebar
* Footer

A beginning html document looks like this:

```html
<!DOCTYPE html>
<html>
   <head>
      <title>MySite</title>
  </head>
  <body>
     <!-- content goes here -->
  </body>
</html>
```


Once you have drawn out your layout in a image editor you can lay out your code as follows:
```html
<body>
    <div id="header">
    </div><!--end header -->
     
    <div id="content">
    </div><!--end content-->
     
    <div id="sidebar">
    </div><!--end sidebar-->
    
    <div id="footer">
    </div><!--end footer-->
     
</body>
```
